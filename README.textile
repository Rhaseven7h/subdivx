h1. SubDivX

h2. Subtitle downloader for http://www.subdivx.com/

p. As simple as that, it let's you search, download and extract subtitles in spanish for divx movies.

h2. Screenshot

p. !http://lh6.ggpht.com/_gswK3oaYqd0/SjACwGRp45I/AAAAAAAAAEU/Y1PKvS92EfA/SubDIvX%20Snapshot.png!

h2. Requirements

p. Packages:

* Ruby
* Gems

p. Gems

* nokogiri
* open-uri
* Ruby-GNOME2

h2. Installation

p. Just unpack, install requirements and run.
